# ♟️ Primeiro Lance

O Primeiro Lance é uma plataforma web desenvolvida como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Desenvolvimento de Sistemas da ETEC Vasco Antonio Venchiarutti.

O projeto tem como objetivo ensinar xadrez de forma interativa, oferecendo conteúdos teóricos, exercícios práticos e acompanhamento da evolução do usuário.

## Objetivos

- Tornar o aprendizado do xadrez mais acessível.
- Auxiliar jogadores iniciantes e intermediarios através de conteúdos organizados.
- Oferecer uma plataforma intuitiva para estudo.
- Registrar e orientar o progresso do usuário.
